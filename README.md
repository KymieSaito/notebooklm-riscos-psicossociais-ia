# Inteligência Artificial e Riscos Psicossociais no Trabalho — NotebookLM + RAG

## Objetivo

Este projeto explora o uso do NotebookLM como ferramenta de aprendizagem ativa para estudo de riscos psicossociais no trabalho, saúde mental ocupacional e IA generativa aplicada à organização do conhecimento.

## Fontes utilizadas

- NR-1
- WHO Mental Health at Work
- ILO Psychosocial Risks
- Artigos sobre IA e RAG

## Ferramentas

- NotebookLM
- GitHub
- IA Generativa
- RAG

## Prompts testados

### Prompt 1
Explique os principais riscos psicossociais relacionados ao ambiente de trabalho.

### Prompt 2
Crie um resumo estruturado sobre NR-1 e saúde mental ocupacional.

## Conclusão

O projeto demonstrou como ferramentas de IA podem auxiliar na aprendizagem ativa, organização de fontes e construção de conhecimento estruturado.
